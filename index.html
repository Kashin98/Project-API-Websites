API List

City Weather

It's a weather app that provides weather of 200,000+ cities across the world. The API is from Open Weather Map. Shows temperature, humidity and the exact type of weather in the city.

Mock Shop

It's an app that adds an Item with an image and name provided by the user, then submits the data to a mock API and displays it immediately. The data is permanent as all submissions are added to mock API. The app allows you to edit or delete any items. Mock API json data can be seen with footer link.

wikipedia search

It's a website that replicates wikipedia search. It is uses the wikipedia API to produce top results that come from keyword that is searched. Max search results are 10.

Chuck Norris jokes

Website that generates a Chuck Norris Joke. Uses Chuck Norris Jokes API to generate multiple prewritten jokes, you get the idea. Use at own risk.

Random person

It generates a random person with personal details using the random user API. All details are fake, I mean I saw a child aged 78 get generated...
